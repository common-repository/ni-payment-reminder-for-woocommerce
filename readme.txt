Automate payment reminders for WooCommerce orders with the Ni Payment Reminder for WooCommerce plugin. Customize reminder emails with shortcodes. Get sales analysis on your dashboard. Contact support for feedback or new report requirements.

== Description ==
<p>
The Ni Payment Reminder for WooCommerce plugin enables you to send payment reminder emails to customers with unpaid orders. With a convenient email reminder button in the WooCommerce Order list, you can easily manage payment reminders. Customize your reminder emails using shortcodes to include order details such as order number, order date, and first name. The plugin also provides settings to customize the sender name, sender email, and subject line of the reminder emails.
</p>
<p>
Customize reminder email with short code like order number, order date, first name and many more. Setting provide the option to set the from name and from email and subject line
</p>


<h4>Key Features of Ni Payment Reminder for WooCommerce</h4>
<ul>
<li>Send payment reminder emails to WooCommerce customers with unpaid orders.</li>
<li>Access the email reminder button directly from the WooCommerce Order list.</li>
<li>Customize payment reminder emails using shortcodes for personalized content.</li>
<li>Get valuable sales analysis on your dashboard, including today's sales, yesterday's sales, and daily sales.</li>
</ul>

== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate "Ni Payment Reminder For WooCommerce" plugin from admin panel.
3. Find "Payment Reminder" menu in admin.
